Instruções para instalação e execução do trabalho.
1. Certifique-se de ter o Java Development Kit (JDK) versão 17.0.7 instalado corretamente em seu sistema. Você pode baixar a versão adequada do JDK em: https://www.oracle.com/java/technologies/downloads/#java17
Caso esteja com problemas na instalação siga os passo do video: https://www.youtube.com/watch?v=_tyw5KZQneo&ab_channel=WendelSegadilha

2. Crie uma pasta chamada "dados" na unidade C do seu computador. Por exemplo, você pode criar a pasta em: C:\dados.

3. Abra o prompt de comando (CMD) e navegue até o diretório onde está localizado o arquivo executável "SistemaControleDeDados.jar". Por exemplo, se o arquivo estiver na área de trabalho, você pode usar o comando cd seguido do caminho completo até o diretório, por exemplo: cd C:\Users\polak\OneDrive\Área de Trabalho\A3 - LuizCunha_e_Jhames.

4. Certifique-se de que o prompt de comando está exibindo o diretório correto onde o arquivo ".jar" está localizado. Você pode verificar isso executando o comando dir e verificando se o arquivo "SistemaControleDeDados.jar" é listado corretamente.

5. Agora, digite o seguinte comando no prompt de comando: java -jar SistemaControleDeDados.jar

6. Se você seguiu todos os passos corretamente, o programa será executado e você poderá usá-lo conforme necessário.

Certifique-se de seguir todas as etapas corretamente para garantir que o programa seja executado com sucesso.
